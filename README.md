# Lemcorian – Global Multidivision Conglomerate

Corporate website for Lemcorian, a diversified multidivision conglomerate operating across Agricultural Products, Chemicals, and Medical Equipment & Instruments sectors.

## Project Overview

This is the official corporate website for the Lemcorian Group, showcasing our global operations, business divisions, leadership team, and corporate information.

## Technologies

This project is built with:

- **Vite** - Build tool and development server
- **TypeScript** - Type-safe JavaScript
- **React** - UI framework
- **shadcn-ui** - Component library
- **Tailwind CSS** - Styling framework
- **React Router** - Client-side routing

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn package manager

### Installation

1. Clone the repository:
```sh
git clone <repository-url>
cd lemcorian-conglomerate-hub
```

2. Install dependencies:
```sh
npm install
```

3. Start the development server:
```sh
npm run dev
```

The website will be available at `http://localhost:8080`

### Build for Production

```sh
npm run build
```

The production build will be generated in the `dist` directory.

### Preview Production Build

```sh
npm run preview
```

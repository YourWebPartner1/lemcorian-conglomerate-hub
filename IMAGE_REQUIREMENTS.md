# Image Requirements for Lemcorian Website

This document lists all premium corporate images needed for the website. All images should be:
- Professional, realistic photography
- Consistent color grading (navy/blue/grey tones)
- Optimized for web (WebP format preferred, JPG acceptable)
- Responsive and properly sized
- Free from watermarks or branding
